Strategic card game inspired by mechanics of a game called Inscryption. The game is settled in modern time military setting and is a competetive strategic game focused on player's ability to create their own synergies and counters. 
Unlike many competetive card games Military Deck provides all content to players with all donation-unlocks being cosmetic items. The game also has flexible unit-based system that facilitates further developing of gamemodes and other systems.
